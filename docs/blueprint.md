# **App Name**: PortfolioFlow

## Core Features:

- Profile Creation: Allow users to input and edit their personal information, work experience, skills, and projects.
- CV Upload & Display: Enable users to upload their CVs in PDF format and display them as part of their profile, or to extract the key info with a tool
- Project Showcase: Create a visually appealing showcase of projects with descriptions, images, and links to live demos or GitHub repositories.
- Skills Visualization: Display skills using charts or graphs for a quick overview of proficiency.
- Contact Form: Implement a contact form for visitors to directly reach out to the portfolio owner.
- Mobile Responsive Design: Ensure the portfolio website is fully responsive and looks great on all devices (desktops, tablets, and smartphones).

## Style Guidelines:

- Primary color: Slate Blue (#778899) for a professional and calming feel.
- Background color: Very Light Gray (#F0F8FF), close to white, for a clean and unobtrusive backdrop.
- Accent color: Steel Blue (#4682B4) to highlight interactive elements and calls to action.
- Font: 'Inter', sans-serif, for a clean, modern, and readable text.
- A clean and structured layout with clear sections for different types of content.
- Use a set of simple, professional icons (e.g., FontAwesome or Material Icons) to represent different skills, technologies, or contact methods.
- Subtle animations to transition between sections and highlight user interactions.
